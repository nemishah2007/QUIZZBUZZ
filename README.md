# QUIZZBUZZ
An interactive quiz application built using HTML, CSS, and JavaScript.
It supports multiple categories, a timer for each question, and real-time score tracking.

Features
Multiple categories:
General Knowledge
Tech
Sports
Entertainment
 Countdown timer for each question
 Instant feedback (correct / wrong / time up)
 Live score tracking
 Automatic navigation to next question
 Final result screen with score & accuracy
Tech Stack
HTML
CSS
JavaScript (Vanilla JS)
