# QUIZZBUZZ

An interactive quiz application built using HTML, CSS, and JavaScript.

---

## Features

* Multiple categories:

  * General Knowledge
  * Tech
  * Sports
  * Entertainment
* Countdown timer for each question
* Instant feedback (correct / wrong / time up)
* Live score tracking
* Automatic next question
* Final result screen

---

## Tech Stack

* HTML
* CSS
* JavaScript

---

## Run Locally

```bash
git clone https://github.com/your-username/quiz-app.git
```

Open `index.html` in your browser.

---

## How It Works

* Questions are stored in a JavaScript object
* Category is selected via URL parameters
* Timer automatically moves to the next question
* Score updates based on correct answers
